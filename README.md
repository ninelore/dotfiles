## Installation
**I recommend using the paru pacman/AUR wrapper.**

To install all requirements on Arch Linux run this: **requires yay**   
(`cd dotfiles`)   
`paru -Syu --needed $(cat requirements.txt)`   
`./install-vimplug-ohmyzsh.sh`
